 Nguyễn_Thị_Ái_Nhi : Hoa_hậu Phụ_nữ Việt_Nam qua ảnh 2005
Từ trái sang : Nguyễn_Trang_Nhung , Nguyễn_Thị_Ái_Nhi , Trương_Thị_Trúc_Diễm_Tối 2-12 , tại nhà_thi_đấu Phan_Đình_Phùng ( TPHCM ) , cuộc_thi “ Hoa_hậu Phụ_nữ Việt_Nam qua ảnh 2005 ” do Báo_Phụ nữ Việt_Nam và Tạp_chí Thế_giới Phụ_nữ tổ_chức , đã kết_thúc sau 6 tháng tranh tài .
Cô Nguyễn_Thị_Ái_Nhi ( sinh_viên , SN 1985 - giữa ) đã trở_thành hoa_hậu và đoạt thêm giải Mặc áo_dài đẹp nhất ; Á_hậu 1 là Trương_Tri_Trúc_Diễm ( và giải Gương_mặt khả_ái - bìa phải ) ; Á_hậu 2 là Nguyễn_Trang_Nhung ( và giải Phong_cách biểu_diễn đẹp nhất - bìa trái ) . Các thí_sinh : Trần_Thị_Hương_Giang ( Mái_tóc đẹp nhất ) ; Lê_Thị_Kim_Thi ( Bạn_đọc bình_chọn ) ; Huỳnh_Thị_Thanh_Tuyền ( Nụ_cười duyên_dáng nhất ) ; Chung_Thục_Quyên ( Ăn ảnh nhất ) ; Lương_Thị_Ngọc_Lan ( Miss E100 ) ; Vũ_Nguyễn_Hà_Anh ( Hình_thể đẹp nhất ) ; Vũ_Thu_Phương ( Mặc áo dạ_hội đẹp nhất ) ...
( Theo Sài_Gòn_Giải_Phóng )
