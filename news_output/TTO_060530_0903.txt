 Ma_thuật file flash với “ Aero SWF.max ”
Gọi_Aero SWF.max có hàng_loạt khả_năng hấp_dẫn tác_động đến file flash như : chuyển file flash thành dạng file EXE tự_động chạy ; khả_năng trình_diễn file flash chuyên_nghiệp ; trích xuất hình_ảnh , âm_thanh từ file flash ; dùng file flash để làm wallpaper , screensave …
Sau đây là một_số tính_năng độc_đáo nhất của Aero SWF.max mà các ứng_dụng chơi flash khác hầu_như không tích_hợp được đầy_đủ được như Aero SWF.max :
- Chơi file flash theo kiểu “ liên khúc ” : đây là tính_năng chơi hàng_loạt file flash liền nhau . Vào menu “ File ” - & gt ; chọn “ Add to Playlist ” để Aero SWF.max chơi hàng_loạt file flash đã nhập vào danh_sách .
- Tạo file flash tự_động chạy : vào menu “ File ” - & gt ; chọn “ Convert to EXE” - & gt ; tìm đến file flash cần chuyển_đổi và nhấn “ Save ” là file SWF sẽ tự_động được chuyển thành dạng file EXE tự_động chạy .
- Thông_tin chi_tiết về file flash : tính_năng “ Get_Info ” ở menu File sẽ thông_báo cho bạn biết toàn_bộ thông_tin về file flash như : dung_lượng , thời_gian chơi , số khung hình / giây , độ_phân_giải hình_ảnh …
- Dùng file flash để làm màn_hình nền : vào menu “ Tool ” - & gt ; chọn “ Flash_Desktop ” - & gt ; chọn “ Enable_Flash_Desktop ” và nhấn OK là file flash của bạn sẽ được chọn làm một dạng màn_hình nền động với đầy_đủ âm_thanh và hình_ảnh chơi liên_tục .
- Dùng file flash để làm trình bảo_vệ màn_hình : vào menu “ Tool ” - & gt ; chọn “ Flash_Screen_Save ” .
- Dùng file flash để làm “ Wallpaper ” : dùng thanh trượt để tìm đến hình_ảnh cần dùng trong file flash - & gt ; vào “ Tool ” - & gt ; chọn “ Set as Wallpaper ” là hình_ảnh đang hiển_thị sẽ được dùng làm Wallpaper ngay lúc đã chọn xong .
- Trích xuất hình_ảnh và âm_thanh trong file flash : vào “ Tool ” - & gt ; chọn “ Extract_Resources … ” . Muốn trích xuất file flash thành hàng_loạt file ảnh JPG thì đánh_dấu vào ô “ JPEG Compressed_Images ” . Muốn trích xuất toàn_bộ âm_thanh trong file flash thành dạng file MP3 thì bạn chọn “ Event MP3 Sounds ” hoặc “ Streaming MP3 Sounds ” .
- Trích xuất một đoạn phim trong file flash : vào menu “ Tool ” - & gt ; chọn “ Capture_Frame ” .
