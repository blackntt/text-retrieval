 Bảo_mật thông_tin với Easy_Hide_Folders 1.0
Đây là một chương_trình bảo_vệ file và folder rất nhỏ gọn nhưng cũng rất hiệu_quả , với giao_diện được thiết_kế đơn_giản , dễ sử_dụng và quản_lý ngay cả đối_với những người mới làm_quen với tin_học .
Chương_trình hỗ_trợ ẩn tất_cả các dạng file và folder trong máy_tính dù chạy chế_độ safe mod hay trên cả hệ_thống mạng cũng không hề hiển_thị được các file hay folder mà chương_trình đã ẩn . Giúp bạn giữ các tài_liệu quan_trọng một_cách dễ_dàng và nhanh_chóng . Ngoài_ra chương_trình còn đặc_biệt hỗ_trợ các file trên định_dạng hệ_thống NTFS , FAT32 , FAT .
- Để thêm một thư_mục vào danh_sách quản_lý bạn click Add_Drectory , lúc này chương_trình hiện ra một cửa_sổ mới ( hình 1 ) . Bạn click vào Browse và chọn thư_mục cần ẩn .
Hình 1
- Để thêm một file vào danh_sách quản_lý bạn click Add_File và thao_tác tương_tự như Add_Drectory .
Trong mục quản_lý File và Folder chương_trình đưa ra các chức_năng : Add_Drectory , Add_File , Enable_Hiding ( ẩn file hoặc folder đã chọn ) , Disable_Hiding ( bỏ ẩn file hoặc folder đã chọn ) , Edit ( chỉnh_sửa đường_dẫn file hoặc Folder trong bảng quản_lý ) , Remove ( Loại_bỏ file hoặc folder ra khỏi danh_sách quản_lý ) , Clear_List ( xóa toàn_bộ danh_sách các file và folder ) .
Để bảo_đảm tính bảo_mật cho các tài_liệu của bạn , bạn nên click vào File à Define_Fassword ( hình 2 ) . Bạn gõ password cần tạo rồi ấn Valid để xác_nhận .
Các bạn chú_ý : chương_trình này được viết chính bằng ngôn_ngữ Pháp , mặc_dù chương_trình có hỗ_trợ tiếng Anh nhưng có một_số mục vẫn giữ nguyên gốc tiếng Pháp .
Hình 2
Chương_trình có dung_lượng khoảng 2.7 Mb tương_thích với hệ_điều_hành Windows 2000/XP và 2003 .
