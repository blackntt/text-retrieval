 Khánh_Hòa : Tăng_giá nước sinh_hoạt
Theo đó , giá nước sinh_hoạt của người_dân được tính theo bốn mức như sau : mức 1 ( tối_đa 5m3/người/tháng ) là 1.911đ/m3 ( tăng 261đ/m3 ) ; mức 2 ( tối_đa 10m3/người/tháng ) : 2.866đ/m3 ( tăng 666đ ) ; mức 3 ( tối_đa 15m3/người/tháng ) : 3.582đ/m3 ( tăng 82đ ) và mức 4 ( mới được đặt thêm , trên 15m3/người/tháng trở_lên ) : 4.776đ/m3 .
Giá nước mới đối_với các cơ_quan_hành_chính , sự_nghiệp và các đơn_vị sử_dụng phục_vụ mục_đích công_cộng ( như nhà_văn_hóa , công_viên ... ) : 2.866đ/m3 ( tăng 366đ ) ; bệnh_viện , nhà_hộ_sinh : 2.388đ/m3 ( tăng 188đ ) ; hoạt_động sản_xuất vật_chất : 3.582đ/m3 ( tăng 82đ ) ; hoạt_động kinh_doanh dịch_vụ , du_lịch : 5.493đ/m3 ( tăng 493đ ) .
