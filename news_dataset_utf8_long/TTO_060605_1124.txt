Săn Media trên mạng hữu hiệu bằng Arachnid

Tuy tốc độ tìm kiếm không nhanh bằng Google, Yahoo… nhưng tính về mức độ hữu dụng và chuyên nghiệp của Arachnid thì tỏ ra hơn hẳn. Khi tìm ra được một kết quả nào đó, Arachnid sẽ liệt kê cho chúng ta đầy đủ mọi thông tin về file media đó như tên file, dung lượng, đường link chi tiết cũng như mọi ghi chú… 
Cách sử dụng Arachnid để săn file media cũng hết sức dễ dàng. Chẳng hạn bạn muốn săn tất cả các file media liên quan đến xuất phẩm điện ảnh nổi tiếng nhất thế giới hiện nay là “Da Vinci Code” thì bạn gõ vào ô tìm kiếm (có biểu tượng cái kính màu xanh) từ khóa là “Da Vinci Code” -&gt; sau đó nhấn vào nút “Search” để Arachnid bắt đầu quá trình tìm kiếm và mọi kết quả sẽ được liệt kê ngay khi Arachnid săn được bất kỳ file nào có liên quan đến từ khóa cần tìm. 
Chỉ trong vòng vài phút, Arachnid đã có thể liệt kê ra cho bạn hàng trăm kết quả liên quan đến “Da Vinci Code” bao gồm hàng trăm file ảnh, file video với đủ định dạng như jpg, gif, mp3, wma, wmv, ram, rm, mov, mp3… Tuyệt vời nhất là Arachnid đã săn đuợc 2 file video dạng WMV và MOV với dung lượng khoảng 70 và 90 MB trong đó có hầu như đầy đủ mọi pha gay cấn nhất của bộ phim “Da Vinci Code”. Bạn chỉ cần nhấp đúp vào file tìm được là Arachnid sẽ đưa bạn ngay đến website chứa file ảnh để xem hoặc kích hoạt ngay chương trình tải file video về máy tính. 
Để tinh lọc quá trình tìm kiếm, bạn có thể bắt Arachnid chỉ tìm kiếm trên mạng G1G2Net (mạng lưu trữ media của riêng Arachnid ) hay tìm toàn bộ trên Internet. Chọn “All types” để bắt Arachnid tìm đầy đủ mọi định dạng file media, còn nếu muốn tìm trong phạm vi hẹp thì bạn lựa những tùy chọn như Image, Audio, Video. 
Ngoài việc săn tin, Arachnid cũng là một công cụ chuyên dùng để chơi đủ loại định dạng file media. Bạn chỉ cần nhấn vào nút Media Library sau đó chọn “Network Media” hoặc “Personal Media” và tìm đến file cần chơi là Arachnid sẽ cung cấp cho bạn một giao diện trình diễn cũng rất chuyên nghiệp. Bạn cũng có thể chơi file media trực tuyến bằng cách nhấp chuột phải vào file media, sau đó chọn “Preview Media File Selected”. 
Arachnid chỉ hoạt động được khi máy tính đã kết nối thành công vào Internet và luôn chạy ở chế độ ngầm định bằng biểu tượng là một con nhện màu trắng nằm ở khay đồng hồ. Tất cả mọi kết quả tìm kiếm đều được Arachnid tự động lưu lại bằng một file HTML nằm trong thư mục cài đặt với đầy đủ mọi thông tin và đường dẫn chi tiết. 
Công ty WebMedia Spider Software đang bán ra bản "Arachnid - Media Search &amp; Play PRO" với giá là 20 USD. Người dùng có thể vào đây để tải về bản dùng thử với dung lượng khoảng 3,64 MB hoặc tìm mua tại các cửa hàng phần mềm.
