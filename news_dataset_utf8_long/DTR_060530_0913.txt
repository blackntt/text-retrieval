Tạo hình nền bằng Flash Wallpaper Macker

Bạn làm việc nhiều trên máy tính và chán cái hình nền lúc nào cũng tĩnh lặng hoặc bạn muốn thay đổi hình nền sống động hơn cho một dịp đặc biệt nào đó? Vậy bạn phải làm sao? Flash Wallpaper sẽ giúp bạn trả lời điều đó. 
 Chương trình Flash Wallpaper Macker giúp bạn tạo ra các hình nền sống động từ chính những Macromedia Flash mà bạn yêu thích. 
Bạn chạy chương trình và vào mục Flash Movie File để chọn file Flash mà bạn muốn tạo trang nền. Vào mục settings (hình 1) và thiết lập các thông số cần thiết. 
+ Movie Quality : Chọn tốc độ nhanh, chậm, hoặc mặc định của hình ảnh flash. 
+ Movie Sizes : Để chọn kích cỡ hiển thị (chương trình cho bạn 3 lựa chọn : Exact Fit, Show All, No Scane) hoặc để trống theo mặc định ban đầu của movie flash. 
+ Các mục Left , Top, Right, Bottom : để các bạn hiệu chỉnh lề, hoặc để theo mặc định. 
Sau đó bạn click vào Preview để xem kết quả, Active Wallpaper để thiết lập hình nền cho máy tính của bạn. 
Hình 1 
Ngoài ra chương trình còn mang đến cho bạn một tính năng độc đáo nữa là tạo tự động thiết lập hình nền bằng việc tạo ra một file tự chạy .exe để bạn có thể gửi hình nền bằng Flash do bạn tạo ra cho bạn bè qua email, forum … 
Để làm được điều đó bạn chỉ cần click vào Create Installer khi bạn tạo xong hình nền bằng Flash. 
Chương trình tương thích với hệ điều hành Windows : 98/Me/2000/XP.
