Xóa dữ liệu vĩnh viễn với File Deleter

Bạn có những file chứa thông tin “tối mật” cần xóa đi một cách vĩnh viễn? Các công cụ phục hồi file bị xóa hiện nay là cực mạnh, có thể “moi” ra file bị xóa ngay khi bạn đã “Empty Recycle Bin” hoặc format đĩa cứng. Sao bạn không thử công cụ mang tên File Deleter một lần?

 File Deleter xóa file triệt để bằng cách vừa xóa vừa ghi đè các dữ liệu vô nghĩa lên phân vùng dữ liệu vừa xóa xong. Đây là công nghệ mang tên DOD do Bộ Quốc phòng Mỹ đang áp dụng cho các máy tính trong hệ thống cơ quan của mình, với số lần ghi đè tối đa có thể lên đến bảy lần, đủ sức qua mặt hầu hết các công cụ phục hồi dữ liệu dạng thương mại hiện nay. 
Cách sử dụng File Deleter cũng rất đơn giản. Bạn vào “Select Files” để chọn những file cần xóa. Các file này sẽ hiện lên trong bản danh sách “File Name” hoặc dùng nút “Clear” để xóa chọn, sau đó nhấn vào nút “Delete Files”. Trước khi xóa, File Deleter sẽ đưa ra cảnh báo rằng “những file đã bị xóa đi sẽ không thể nào phục hồi lại được” để xác định ý chí xóa file của bạn. Khi nhấn vào “Yes” là bạn có thể yên tâm rằng những file đã xóa sẽ không còn hiện diện trên cõi đời này nữa. 
File Deleter áp dụng qui trình xóa file trực tiếp nên bạn đừng mong mỏi tìm lại những file đã xóa trong “Recycle Bin”. Qua thử nghiệm, các công cụ phục hồi dữ liệu phổ biến như Recover My Files, File Recover Professional, Acronis Recovery Expert… đều gần như bó tay trước File Deleter. 
Công ty DigitByte Studio đang bán công cụ này với giá 20 USD. Người dùng có thể vào đây để tải về bản dùng thử trong 30 ngày với dung lượng rất mini, chỉ 420 KB. Hiện File Deleter đã có mặt tại các cửa hàng phần mềm tin học.
