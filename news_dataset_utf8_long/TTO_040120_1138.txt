Gửi e-mail chúc Tết sướng mắt, đã tai

Bác Bill Gates nhà ta có “ém” trong công cụ e-mail miễn phí Outlook Express của mình cái chức năng tạo ra những bức thư điện tử multimedia thật là sướng mắt, đã tai người nhận. Vấn đề chỉ còn tùy thuộc vào óc sáng tạo và khả năng vẽ vời của bạn. 
Trong dịp Tết Giáp Thân cầm tinh của Tề Thiên Đại thánh này, bạn thử trổ tài gửi những bức “e-mail có cánh” tới bạn bè, người thân của mình xem sao nhé. Trước hết, bạn chuẩn bị vật liệu sẵn bằng cách copy các file hình ảnh, âm thanh vào một thư mục tạm nào đó. 
A. SOẠN THƯ: 
1. Mở Outlook Express (OE). 
2. Nhấn vào nút Create Mail hay chọn New/ Mail Message trong menu File. 
3. Trong dòng To, bạn gõ địa chỉ e-mail người nhận vào. Nếu địa chỉ đó có trong sổ địa chỉ Address Book, bạn nhấn lên phím To để mở cửa sổ Select Recipients (chọn người nhận) lên. Chọn địa chỉ người nhận trong ô Name và nhấn nút To để nạp nó vào e-mail. Trong trường hợp muốn gửi cho nhiều người, bạn nạp địa chỉ vào dòng Cc (mỗi địa chỉ cách nhau bằng dấu chấm phẩy (;). Còn nếu muốn gửi “lén” thêm cho ai đó, bạn dùng dòng Bcc (người nhận chính sẽ chẳng hề hay biết bạn cùng gửi thư này cho những người khác!). Xong, nếu đang xài Address Book, bạn nhấn OK để thoát khỏi cửa sổ Select Recipients. 
4. Trong dòng Subject, bạn gõ tiêu đề của thư. Có thể gõ bằng tiếng Việt Unicode (nhưng nếu máy của người nhận không hỗ trợ Unicode, tỉ dụ như xài Windows 95/98 thì tiêu đề này sẽ bị tiêu tùng, hiện lên toàn mã trông mà bắt ghét). 
5. Gõ nội dung lời chúc Tết vào thân thư. Bạn có thể chọn font, cỡ chữ, kiểu chữ, màu sắc,... Thậm chí nếu có máu màu mè hoa lá cành, bạn có thể thà sức mà vẽ vời, như cho mỗi chữ một kiểu, một màu. 
B. TÔ ĐIỂM CHO THƯ: 
Bây giờ mới tới màn bạn thêm mắm, dậm muối các món nghe nhìn vào e-mail. 
Hình ảnh: Để đưa hình ảnh vào thư, bạn nhấn menu Insert và chọn Pictures. 
Trên đó, bạn nhấn nút Browse để mở tới thư mục chứa ảnh muốn nạp vào thư. Hỗ trợ các định dạng file ảnh JPG, GIF, BMP, WMF (Windows Metafile), XBM, ART,... Bạn chỉ nên dùng định dạng JPG hay GIF cho file nhẹ. Để thêm ấn tượng, bạn có thể dùng file AniGIF để khi mở lên hình ảnh nó “cục cựa” cho vui mắt. Ở mục Alternate Text, bạn gõ vào nội dung mà mình muốn nó xuất hiện dưới dạng Tool Tip mỗi khi người mở rê chuột lên ảnh. Phần Layout để bạn chọn vị trí ảnh nằm trên thư (Alignment, nhấn nút mũi tên để chọn trong list), độ dày của khung ảnh (Border Thickness, gõ vào kích thước, số càng lớn, khung càng dày, nhưng không nên chọn khung quá dày vì là khung đen), vị trí ảnh nằm cách các đối tượng khác (Horizontal: ngang, Vertical: dọc). Xong thì nhấn OK. 
Âm thanh: Bạn nạp thêm file nhạc để khi người ta mở thư, tiếng nhạc sẽ trỗi lên cho khoái cái lỗ tai. Nhấn vào menu Format, chọn Background/Sound.... 
Hộp thoại Background Sound xuất hiện. Bạn nhấn nút Browse mở tới thư mục chứa file âm thanh muốn nạp vào. Hỗ trợ các file Audio (WAV,WMA...) và MIDI. Mặc định nó sẽ mở các file Audio, nên muốn trong danh sách hiển thị các file MIDI, bạn phải xác định dạng Types of file là MIDI files. Bạn có thể xác định cho file nhạc đó được chơi bao nhiêu lần (Play the sound.... times, chọn từ 1 tới 999 lần) hay chơi miết không nghỉ (đánh dấu chọn Continuously). Nhấn OK. Ở đây, xin bỏ nhỏ: bạn có thể tự thu âm lời chúc Tết của mình để gửi kèm vào trong thư (nhưng nên chọn định dạng WMA cho nó nhẹ). 
TRANG TRÍ NỀN THƯ: 
Bạn có thể trang trí nền thư bằng cách nhấn menu Format và chọn Background. Ở đây, bạn có thể làm nền bằng ảnh (chọn Pictures) hay màu sắc (chọn Color). 
Vậy là xong à há. Bạn kiểm tra và save lại e-mail chúc Tết “sướng mắt, đã tai” để rồi có thể gửi đi cho ai đó kìa ta!
